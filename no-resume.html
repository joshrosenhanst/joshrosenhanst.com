TwitchRandom

Twitch.tv is an online, live streaming service. People use the service to stream their video game sessions.

TwitchRandom is a project that finds random streams on Twitch to watch, by using Twitch's public API. TwitchRandom can provide completely random streams or filter random streams by game.

The website also works as a testbed for Google's responsive advertising. The site is fully responsive, and the ads resize as needed to fit the layout.

Technologies: HTML5, CSS3, jQuery, Bootstrap, PHP, Laravel

Dota2Layout

Dota 2 is an online game in which the player can select from over 100 characters for each match. The characters are selected through a grid like interface or layout which the players can edit and rearrange in-game.

Dota2Layout is a project that takes that in-game layout and emulates it into a web service, so that players can change their character select layout outside of the game.

The website takes an exported text configuration file from the Dota 2 game and translates the custom coordinate system into values that CSS and javascript can interact with. Once the player is done editing their layout, the website translates the values back into the original coordinate system so that it can be imported back into the game.

The website's layout interface uses custom jQuery event code to allow for dragging and dropping, as well as multi-selection, and a dynamic snap-to-grid system.

Users can also create a Wordpress account, save their edits online, and share their layouts through a community gallery.

Technologies: HTML5, CSS3, jQuery, PHP, Wordpress
